## **customtools 端点说明**

对于使用 **bash 与自定义工具混合构建应用** 的用户，

Gemini 3.1 Pro 提供单独端点：

gemini-3.1-pro-preview-customtools

该端点更擅长优先处理自定义工具（例如 view\_file、search\_code）。

> 注意：虽然 customtools 针对使用自定义工具和 Bash 的代理工作流进行了优化，但在某些无法从这些工具中受益的用例中，可能会出现质量波动。
